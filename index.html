Getting Started

Unzip the downloaded folder and find out the assets. There are all the template files in this folder. You can open the following files for viewing in browser.

  1. css — folder with css files.
  2. images — folder with image files and png Icon.
  3. js — folder with Javascript files.
  4. index.html — the Home page.

There is another folder called bower_components. You will get all the Libraries used in the template in this folder.

Countdown Section

In index.html file . HTML structure of Countdown Section is following. To change the countdown time open assets/js/main.js file. Go to line 175 and put your desired target date just changing the value of day, month and year.

// Countdown jquery

$(function () {
    var day = 31,
        month = 12,
        year = 2016;
    var austDay = new Date(year, month - 1, day);
    $('#defaultCountdown').countdown({until: austDay});
    $('#year').text(austDay.getFullYear());
});
